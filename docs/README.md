## William Ruel‑Lian Ting — Personal Homepage

How to use:

1. Place a professional headshot at `assets/headshot.jpg` (recommended 800×800, square).
2. Install dependencies: `npm install`
3. Build TypeScript: `npm run build`
4. Open `index.html` in a browser or run `npm run serve` for local development.
5. Edit content directly in `index.html`. Styles are in `styles.css`; behavior in `script.ts`.

Development:
- Edit TypeScript in `script.ts`
- Run `npm run watch` for auto-compilation during development
- Run `npm run dev` to build and start a local server

Structure:
- Top navigation with sections: Home, Skills, Projects, Talks, Publications, CV.
- Minimalist academic layout, responsive for mobile and desktop.
- TypeScript for type-safe JavaScript development.
